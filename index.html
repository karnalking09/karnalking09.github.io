Knowledge Base

SQL Injection Payloads

  • Simple: ' OR '1'='1 --
  • Union Based: ' UNION SELECT null,null,null --
  • Blind Boolean: ' AND 1=1 --
  • Time Based: ' OR IF(1=1,SLEEP(5),0) --

Defense: Always use parameterized queries (prepared statements).

Cross-Site Scripting (XSS)

  • Basic: <script>alert(1)</script>
  • Attribute: " onmouseover="alert(1)
  • SVG: <svg onload=alert(1)>

Defense: Output encode + Content-Security-Policy.

Other Vulnerabilities

  • 🗂 IDOR — Insecure Direct Object Reference
  • 🌐 SSRF — Server Side Request Forgery
  • 📂 LFI / RFI — Local/Remote File Inclusion
  • 🔑 Broken Authentication / Session Issues
  • ⚡ RCE — Remote Code Execution

Best Practices

  • ✅ Test only in legal scope
  • ✅ Record request/response proof
  • ✅ Report with steps + remediation
  • ❌ Don’t share live exploit code
